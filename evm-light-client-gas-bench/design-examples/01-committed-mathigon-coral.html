The recommendation

Use secp256k1eth.
Today, that's the answer.

CometBFT validators sign ed25519 over per-signer canonical-vote bytes. Solidity has no ed25519 precompile, so verifying current commits on Ethereum L1 is not viable at quorum scale. Switch the signing curve to secp256k1, run ecrecover per signer, and the verify cost stays native to the EVM.

Slide 06 · The wall

If you keep per-signer messages, BLS-D pays N+1 pairings.

Each signer's message gets its own EIP-2537 pairing check at ~32,600 gas. The line is straight; the slope is steep. By N=175 you're at 5.8M gas for verification alone. Every Geth block has a 30M ceiling.
