# Direction probes

Four small static HTML files probing color strategy and reference DNA for the explainable refactor. Each file shows four slides — cover plus three demonstrative beats — so the rhythm reads like a deck, not just a hero. One color strategy each (Restrained / Committed / Full palette / Drenched), one anchor reference each (Stripe Press / Mathigon / 3Blue1Brown / Pudding).

Open any of them directly in a browser.

## 01 — Committed × Mathigon (warm coral)

`01-committed-mathigon-coral.html`

- **Strategy:** Committed. Tinted-cream surface; one warm coral accent carries hero italics, the recommendation moment, and the formula tokens.
- **Reference:** Mathigon / Bartosz Ciechanowski hybrid. Iowan Old Style display italic, mono labels and rails, intersection-style measurement chart.
- **Slides:** cover (verdict) · the gap (annotated flow with broken-node diagram) · the number (italic display 295,855 with breakdown) · the wall (animated formula + line chart with measured dots that pop in, BLS-D slider).
- **What it commits to:** restraint everywhere except the moments that matter. The coral only appears where the argument lands.

## 02 — Drenched × Pudding (warm acid)

`02-drenched-pudding-acid.html`

- **Strategy:** Drenched cover, calm body. The cover IS the color (warm acid yellow-green); subsequent slides drop to a pale neutral so the data essay can breathe.
- **Reference:** Pudding / data essay. Each scheme has its own permanent color (secp coral, BLS-A teal, BLS-B indigo, BLS-C jade, BLS-D brick) and that color follows it across every slide.
- **Slides:** drenched cover (italic display + pull-quote highlights) · the gap on calm (small-multiples flow with brick-coded constraint rows) · the recommendation (italic display number with secp-coral side accent) · comparison race-bars (animated, with conditional / recommended badges).
- **What it commits to:** a strong opening fold that earns attention; data slides that respect the chart by getting out of its way.

## 03 — Full palette × 3Blue1Brown

`03-fullpalette-3b1b.html`

- **Strategy:** Full palette. Five named colors, one per scheme. The schemes get introduced as named characters on the cover; each color persists across every slide.
- **Reference:** 3Blue1Brown explainable cadence. Rounded display family (SF Pro Rounded / system rounded), characters-with-glyphs to humanize the schemes, hero number on a tinted scheme-color background.
- **Slides:** palette cover (the five characters drift in staggered) · the gap (schemes lined up against a hatched EVM-constraint wall, each in its color) · hero number (secp's verdict on a coral-tinted backdrop) · the field (race bars with all five colors, recommended row tinted, BLS-D bar overflows the chart).
- **What it commits to:** color is doing semantic work, not decoration. The reader learns "BLS-B is the indigo one" on slide 1 and that holds for the rest.

## 04 — Restrained × Stripe Press

`04-restrained-stripe-press.html`

- **Strategy:** Restrained. Cream paper, deep ink, one muted brick accent under 10% of surface — eyebrows, italic emphasis on "secp256k1eth", and the recommended row in the comparison ledger. Otherwise the surface is quiet.
- **Reference:** Stripe Press. Bookish reading column with generous margins, italic serif headlines, drop-cap, marginal notes, footnoted body, ledger-style tabular numerals.
- **Slides:** cover (italic verdict, lots of breathing room) · the gap (essay with drop-cap and marginal note on precompiles) · the number (display tabular numeral with scholarly framing) · the field (small-caps ledger table — no bars, typography and spot color do the work).
- **What it commits to:** authority through restraint. The page reads like a book, not a deck. The brick spot color earns its appearance.

## How to compare

Open all four. Things to feel for:

- **Voice.** Does it sound like the engineer it's for? (Confident, prescriptive, doing real explanatory work.)
- **Color load.** Does the color carry the argument or just decorate? Does any of them feel SaaS-y or crypto-aesthetic?
- **Recommendation legibility.** On the cover, is the verdict clear in three seconds?
- **Chart respect.** When data appears, does the surface help or fight it?
- **Rhythm.** Does the four-slide arc feel like a deck or a brochure?
- **AI-slop test.** Could you guess "AI made this" within five seconds on any of them?

The deck refactor will commit to one direction (or a deliberate hybrid). These probes are not the brief; they're the visual lane test underneath it.
