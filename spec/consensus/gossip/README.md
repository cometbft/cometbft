## Conventions

- MUST, SHOULD, MAY... are used according to RFC2119.
- [X-Y-Z-W.C]
    - X: What
        - VOC: Vocabulary
        - DEF: Definition
        - REQ: Requires
        - PROV: Provides
    - Y-Z: Who-to whom
    - W.C: Identifier.Counter
